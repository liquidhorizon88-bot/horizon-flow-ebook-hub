## Capture before drafting

- Resident name and unit: `[name / unit]`
- Vehicle and visitor details: `[guest name / plate / make-model-color / visit dates / expected arrival]`
- Request type: `[visitor pass / temporary permit / overnight guest / replacement hangtag / parking exception / tow release / accessible-space question / other]`
- Source records to check: `[lease/rules addendum / parking policy / permit log / towing vendor rules / accessible accommodation process / HOA/building rules / prior warnings]`
- Risk or enforcement status: `[tow warning / ticket issued / unauthorized spot / no available guest spaces / expired permit / emergency/service vehicle]`
- What is not verified yet: `[eligibility / fee / enforcement exception / accessible accommodation / tow release / availability / approval owner]`

## STOP AUTOMATION

Do not promise a parking permit, enforcement exception, tow release, fee waiver, refund, accessible-space decision, overnight guest allowance, grace period, or assigned-space change until policy/source records and manager/legal/accommodation review are complete.

## Safe reply snippet

Hi [Name] — thanks for the parking permit request. I’m checking the parking policy, permit log, and any property-specific rules before confirming eligibility, timing, or next steps. I’ll follow up with the approved option or route it for manager review if needed.
