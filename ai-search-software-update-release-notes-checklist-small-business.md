# AI Search Software Update and Release Notes Checklist for Small Businesses

SOFTWARE-UPDATE-RELEASE-NOTES-CARD-READY

Use this checklist before AI search tools, help-center drafts, chatbots, or support agents answer questions about software updates, firmware versions, release notes, patches, compatibility, rollback, known issues, or support windows.


## Safe reply snippets

- **Feature/version question:** I need to check the release-note source and your product/version before confirming availability, so we do not promise a feature that is not in your current plan or build.
- **Rollback question:** Rollback options depend on the approved support policy and the version you are running. I’ll verify the source before giving instructions or timing.
- **Bug-fix question:** We should compare your case with the release note, known-issue list, and support record before claiming the update fixed it.
