20-minute Friday review agenda

  1. Five minutes: count new leads by phone, form, Google message, social/chat, referral, and marketplace.
  2. Five minutes: sort by response delay and mark every lead that missed the target window.
  3. Five minutes: compare booked jobs by source so the team does not optimize for noisy inquiry volume.
  4. Three minutes: pick one broken handoff: unassigned owner, late reply, unclear quote next step, missing photo request, or stale follow-up.
  5. Two minutes: assign one improvement to one person for next week.

Pair this with the speed-to-lead benchmark so the dashboard has realistic target windows.

Owner-safe AI summary prompt

You are helping a local service business review a weekly lead response dashboard. Use only the rows below. Do not invent missing numbers. Identify: (1) the lead source with the biggest response delay, (2) the source with the best booked-job rate, (3) follow-up tasks that appear overdue, (4) one script or routing rule to test next week, and (5) any missing data that prevents a confident decision. Keep customer details anonymized. Dashboard rows: {paste anonymized weekly dashboard data}.

Human-review rule: AI can summarize the dashboard, but a manager must verify source counts, booked revenue, customer opt-out requests, warranty/refund issues, safety-sensitive jobs, and staff performance notes before acting.

What to do with the answer

  • If response time is slow, install a first-reply script before adding more ad spend.
  • If follow-ups are overdue, assign ownership and use a two-touch sequence instead of open-ended chasing.
  • If one source books better, protect it with faster callbacks and clearer intake questions.
  • If no source data exists, start with the lead source tracking spreadsheet guide.
  • If staff performance is sensitive, review privately and improve the system before blaming one person.
