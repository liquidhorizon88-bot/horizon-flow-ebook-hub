Rental Application Adverse Action Source Checklist for Property Managers

Use this worksheet before an AI draft, leasing assistant, or applicant inbox replies to a rental applicant who asks why an application was denied, conditionally approved, delayed, rerouted to a co-signer, or sent for additional screening review.

When to use it

Best fit: property managers, leasing coordinators, and applicant-support teams that need to acknowledge application-outcome questions without inventing denial reasons, credit-score details, screening criteria, legal notices, approval odds, or exception promises.

Outcome: a documented source card, applicant-safe acknowledgement, and compliance/legal review stop rule before anyone uses AI to summarize, explain, or answer adverse-action or conditional-approval questions.

Adverse-action source card

  • Application/prospect ID: [application ID / prospect record]
  • Property/unit or floor plan: [property / unit / floor plan]
  • Applicant question type: [denial reason / conditional approval / co-signer / extra deposit / pending screen / appeal / document request]
  • Official screening source: [screening provider record / rental criteria / adverse-action notice / compliance-approved template]
  • What has been sent already: [notice date / method / template ID / portal message / certified mail / email]
  • What is not verified yet: [specific reason / score factor / exception authority / appeal path / timing / eligibility]
  • Required reviewer: [leasing manager / screening provider / compliance/legal / fair-housing reviewer]

STOP AUTOMATION rules

  • Do not invent or paraphrase denial reasons. Use only the approved notice, screening-provider output, and manager/compliance guidance.
  • Do not disclose sensitive screening details casually. Credit, criminal, income, eviction, identity, fraud, or background-screening content needs the approved channel and notice process.
  • Do not promise approval, appeal results, fee changes, deposit changes, or move-in timing. Route any exception, reconsideration, accommodation-sensitive context, or conditional approval question to the named reviewer.
  • Do not make case-by-case criteria changes in an AI reply. Fair-housing consistency and local legal requirements must be reviewed before sending applicant-specific outcome wording.
  • Do not let AI summarize protected-class-sensitive or legal-sensitive facts. If the applicant mentions disability, source of income, family status, assistance animal, domestic violence, immigration status, or other protected/sensitive context, stop and escalate.

Applicant-safe reply snippets

Outcome question acknowledgement

Hi [Name] — I received your question about the application outcome. I do not want to guess or summarize the screening decision incorrectly, so I’m checking the official application record and approved notice before sending the verified next step.

Conditional approval / co-signer question

Hi [Name] — conditional approval steps can depend on the official screening record and current rental criteria. I’m routing this for review so any next-step wording matches the approved policy and notice language.

Appeal or reconsideration question

Hi [Name] — thanks for asking. I’m checking the approved process for reconsideration or corrected information before I describe any options. I’ll follow up through [official channel] once that source is verified.

Missing notice / document request

Hi [Name] — I’m checking whether the required notice or document request has already been sent and which official channel should be used. I’ll avoid sending partial details until the record is confirmed.

CRM labels and manager review prompt

  • application_outcome_question
  • adverse_action_notice_review
  • conditional_approval_question
  • screening_provider_source_needed
  • compliance_fair_housing_review

Manager review prompt: “Verify the screening-provider record, rental criteria, adverse-action notice, official communication channel, protected-class-sensitive context, and compliance/legal routing before sending any denial, condition, appeal, fee, deposit, approval, or reconsideration wording.”
